# Алгоритмы

# Задание

Выведите алгоритм решения следующей задачи: "Дано 2 списка с равным количеством чисел. Вычислите попарную сумму элементов из этих списков (первый элемент из первого списка, первый из второго; второй элемент из первого списка, второй из второго; ...)"

# 1. Рассуждения

Прочитав задание можно понять что нужно создать 2 списка с равным количеством чисел. 
Далее вычислить попарную сумму элементов их этих списков. 
Так же нужен еще один список ддя того чтобы туда записасывались значения сумм двух списков.

# 2. Подготовка входных данных

1) Объявить два списка одинаковой длинны.
2) Заполнить оба списка числами.

# 3. Подготовка списка для результата

1) Создать третий список той же длинны для сохранения сумм.
2) Проверить что его длинна соответствует длинам входных списков.
3) Если нет, дальше алгоритм выполнятся не может.

# 4. Организация цикла по элементам списков

1) Создать переменную index для хранения текущего положения. 
2) Установить index = 0.
3) Проверить условие продолжения цикла:
        - Если index < длины списков, выполнять тело цикла.
        - Если index равен длине списков, то цикл завершается.

# 5. Выполние тела цикла (сложение элементов)

1) Взять элемент первого списка по текущему индексу.
2) Взять элемент второго списка по тому же индексу.
3) Сложить эти два числа.
4) Записать полученную сумму в список результатов.

# 6. Переход к следующей итерации

1) Увеличить значение index на 1.
2) Вернуться к шагу 4 и снова проверить условие цикла.

# 7. Вывод

1) Когда цикл завершился, пройти по списку результатов.
2) Вывести каждую сумму подряд на экран.
